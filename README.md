# Trabalho1-FSE

## Como compilar

Apenas rode o comando.

```shell
make
```

## Como rodar:

Apenas rode o comando.

```shell
make run
```

## Log

O log será armazenado no diretório ./log/log.csv

## Curva reflow

Caso queira adicionar um arquivo personalizado de curva reflow, adicione no diretório ./assets/reflow.csv
